# Job Application Tracker - Chrome Extension

A Chrome extension that automatically extracts job application data from job boards and saves it to your Job Application Tracker web app.

## üöÄ Features

- **Auto-extract job data** from supported job boards (Glassdoor, Indeed, etc.)
- **One-click save** to your job application tracker
- **Manual job entry** when auto-extraction isn't available

## Ô∏è Installation

1. **Download the extension:**
   - Download the entire extension folder from the provided Google Drive link.
   - Save the folder to your computer

2. **Load into Chrome:**
   - Open Chrome and navigate to `chrome://extensions/`
   - Enable **"Developer mode"** (toggle in top right)
   - Click **"Load unpacked"**
   - Select the extracted folder app-tracker-extension

## üìñ How to Use

1. **Install the extension** (see installation steps above)
2. **Open the Job Application Tracker web app** at `http://my-app-tracker.s3-website-us-east-1.amazonaws.com`
3. **Log in** to your account
4. **Navigate to a job posting** on a supported site (Glassdoor, Indeed, etc.)
5. **Click the extension icon** in your Chrome toolbar
6. **Click "Extract Job Data"** to automatically fill in job details
7. **Review and save** to add to your tracker

## üîç Troubleshooting

**"Extension Not Ready" Error:**
- Refresh the job posting page after logging in and try again

**"Login Required" Message:**
- Make sure you're logged into the web app
- Click "Go to Login" in the extension to open the web app
